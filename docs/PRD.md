# 노션 기반 북리뷰 MVP PRD

## 🎯 핵심 정보

**목적**: 사용자가 노션에서 작성한 책 리뷰를 웹 서비스에서 통합 관리하고 개인 라이브러리로 열람 및 수정할 수 있도록 함

**사용자**: 책을 자주 읽으며 리뷰를 남기고 싶은 개인 사용자

---

## 🚶 사용자 여정

```
1. 랜딩 페이지
   ↓ [로그인/회원가입 선택]

   [신규 사용자] → 회원가입 페이지 → 계정 생성 → 로그인
   [기존 사용자] → 로그인 페이지 → 인증 완료
   ↓

2. 노션 연동 설정 페이지
   ↓ [노션 API 연동]

   [연동 성공] → 리뷰 목록 페이지 (자동 동기화)
   [연동 실패] → 에러 메시지 표시 → 재시도 옵션
   ↓

3. 리뷰 목록 페이지 (메인 대시보드)
   ↓ [리뷰 조회/관리]

   • 리뷰 조회 → 리뷰 상세 페이지
   • 리뷰 수정 → 리뷰 수정 페이지
   • 리뷰 삭제 → 확인 후 삭제 → 목록 새로고침
   • 새 리뷰 작성 → 리뷰 작성 페이지
   ↓

4. 리뷰 작성/수정 페이지
   ↓ [내용 입력 및 저장]

   [저장 성공] → 리뷰 목록 페이지 (목록 업데이트)
   [저장 실패] → 에러 메시지 표시 → 재시도
   ↓

5. 리뷰 상세 페이지
   ↓ [전체 내용 열람]

   • 수정 → 리뷰 수정 페이지
   • 삭제 → 확인 후 삭제 → 리뷰 목록 페이지
   • 목록으로 → 리뷰 목록 페이지
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F001** | 사용자 회원가입 | 이메일과 비밀번호로 신규 계정 생성 | 서비스 이용의 기본 전제 | 회원가입 페이지 |
| **F002** | 사용자 로그인 | 이메일과 비밀번호로 기존 계정 로그인 | 서비스 이용 권한 관리 | 로그인 페이지 |
| **F003** | 노션 API 연동 | 노션 계정 인증을 통해 공유 데이터베이스 접근 | 북리뷰 데이터 동기화의 핵심 | 노션 연동 설정 페이지 |
| **F004** | 리뷰 목록 조회 | 노션에서 동기화된 모든 책 리뷰를 목록으로 표시 | 사용자의 리뷰 라이브러리 역할 | 리뷰 목록 페이지 |
| **F005** | 리뷰 상세 조회 | 선택한 리뷰의 전체 내용 및 메타데이터 확인 | 리뷰 내용 열람의 기본 기능 | 리뷰 상세 페이지 |
| **F006** | 리뷰 작성 | 새로운 책 리뷰를 직접 작성 및 저장 | 웹 서비스에서 네이티브 리뷰 생성 | 리뷰 작성 페이지 |
| **F007** | 리뷰 수정 | 기존 리뷰의 내용을 변경 및 저장 | 리뷰 내용 관리의 기본 기능 | 리뷰 수정 페이지, 리뷰 상세 페이지 |
| **F008** | 리뷰 삭제 | 불필요한 리뷰 영구 삭제 | 리뷰 라이브러리 관리 기능 | 리뷰 목록 페이지, 리뷰 상세 페이지 |

### 2. MVP 필수 지원 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F009** | 사용자 로그아웃 | 현재 세션 종료 및 로그인 페이지로 리디렉션 | 계정 보안 관리 | 모든 인증된 페이지 |
| **F010** | 노션 자동 동기화 | 일정 시간마다 노션 데이터베이스와 동기화 | 노션 변경사항 실시간 반영 | 리뷰 목록 페이지 |

- **Next.js 15** (App Router) - React 풀스택 프레임워크, 최신 성능 최적화
- **TypeScript 5.6+** - 타입 안전성 및 개발 생산성 보장
- **React 19** - UI 라이브러리 (최신 동시성 기능 및 개선된 훅)

### 🎨 스타일링 & UI

- **TailwindCSS v4** (설정파일 없는 새로운 엔진) - 유틸리티 CSS 프레임워크
- **shadcn/ui** - 고품질 Radix UI 기반 React 컴포넌트 라이브러리
- **Lucide React** - 일관된 아이콘 라이브러리

### 📝 폼 & 검증

- **React Hook Form 7.x** - 경량 폼 상태 관리
- **Zod** - TypeScript 기반 스키마 검증 라이브러리

### 🔐 인증 & 데이터베이스

- **Supabase** - BaaS (인증, PostgreSQL 데이터베이스, 실시간 구독)
- **PostgreSQL** - 관계형 데이터베이스 (Supabase 포함)
- **@supabase/supabase-js** - Supabase JavaScript 클라이언트

### 🔗 외부 API 통합

- **@notionhq/client** - 공식 노션 API 클라이언트
- **node-cache** - 노션 동기화 데이터 캐싱

### 🚀 배포

- **Vercel** - Next.js 15 최적화 배포 플랫폼

### 📦 개발 도구

- **npm** - 의존성 관리
- **ESLint** - 코드 품질 관리
- **Prettier** - 코드 포맷팅
- **Husky + lint-staged** - Git 훅 기반 자동 린트
